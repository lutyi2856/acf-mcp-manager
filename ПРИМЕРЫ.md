# Примеры использования ACF Backend Manager

## Создание страниц опций через MCP

### Пример 1: Базовая страница настроек

```json
{
  "page_title": "Настройки сайта",
  "menu_slug": "site-settings",
  "menu_title": "Настройки",
  "icon_url": "dashicons-admin-settings"
}
```

### Пример 2: Использование готового шаблона

```json
{
  "page_title": "Контактная информация",
  "menu_slug": "contacts",
  "template": "contact_info"
}
```

Доступные шаблоны:
- `general_settings` - Общие настройки
- `theme_settings` - Настройки темы
- `contact_info` - Контактная информация
- `seo_settings` - SEO настройки
- `header_footer` - Шапка и подвал

### Пример 3: Страница с подстраницами

```json
{
  "page_title": "Настройки темы",
  "menu_slug": "theme-main",
  "icon_url": "dashicons-admin-appearance",
  "sub_pages": [
    {
      "page_title": "Цвета и стили",
      "menu_slug": "theme-colors",
      "parent_slug": "theme-main"
    },
    {
      "page_title": "Типографика",
      "menu_slug": "theme-typography",
      "parent_slug": "theme-main"
    },
    {
      "page_title": "Логотипы",
      "menu_slug": "theme-logos",
      "parent_slug": "theme-main"
    }
  ]

```

### Пример 4: Добавление подстраницы к существующей странице WordPress

```json
{
  "page_title": "Дополнительные настройки",
  "menu_slug": "extra-settings",
  "parent_slug": "options-general.php"
}
```

Популярные parent_slug:
- `options-general.php` - Настройки
- `themes.php` - Внешний вид
- `edit.php` - Записи
- `edit.php?post_type=page` - Страницы
- `tools.php` - Инструменты

## Работа через REST API

### Создать страницу опций

```bash
curl -X POST http://localhost:8000/wp-json/acf-cpt-manager/v1/options-pages \
  -H "Content-Type: application/json" \
  -u admin:password \
  -d '{
    "page_title": "Настройки сайта",
    "menu_slug": "site-settings",
    "icon_url": "dashicons-admin-settings"
  }'
```

### Получить список страниц

```bash
curl http://localhost:8000/wp-json/acf-cpt-manager/v1/options-pages \
  -u admin:password
```

### Получить шаблоны

```bash
curl http://localhost:8000/wp-json/acf-cpt-manager/v1/options-pages/templates \
  -u admin:password
```

### Активировать/деактивировать страницу

```bash
curl -X POST http://localhost:8000/wp-json/acf-cpt-manager/v1/options-pages/site-settings/toggle \
  -H "Content-Type: application/json" \
  -u admin:password \
  -d '{"active": false}'
```

## Создание групп полей (Field Groups) через MCP

### РЕКОМЕНДУЕМЫЙ СПОСОБ: target_type + target_value

Используйте `target_type` и `target_value` для указания места отображения группы полей.

**Доступные значения target_type:**
- `post_type` - для типов записей (post, page, doctor, service и др.)
- `taxonomy` - для таксономий/категорий (category, service_category и др.)
- `options_page` - для страниц опций ACF
- `page_template` - для шаблонов страниц
- `user_form` - для форм пользователя
- `nav_menu` - для меню навигации
- `block` - для ACF блоков

### Пример 1: Группа полей для таксономии (категорий)

```json
{
  "title": "Настройки категории услуг",
  "target_type": "taxonomy",
  "target_value": "service_category",
  "fields": [
    {
      "name": "category_image",
      "label": "Изображение категории",
      "type": "image",
      "return_format": "array",
      "preview_size": "medium"

      "name": "category_icon",
      "label": "Иконка",
      "type": "text",

    }
  ]
}
```

## Типы полей ACF

### Базовые
- `text` - Текстовое поле
- `textarea` - Многострочный текст
- `number` - Число
- `range` - Диапазон
- `email` - Email
- `url` - URL
- `password` - Пароль

### Контент
- `image` - Изображение
- `file` - Файл
- `wysiwyg` - Визуальный редактор
- `oembed` - Встраиваемый контент
- `gallery` - Галерея

### Выбор
- `select` - Выпадающий список
- `checkbox` - Флажки
- `radio` - Переключатели
- `button_group` - Группа кнопок
- `true_false` - Да/Нет

### Связи
- `link` - Ссылка
- `post_object` - Объект записи
- `page_link` - Ссылка на страницу
- `relationship` - Связь
- `taxonomy` - Таксономия
- `user` - Пользователь

### Дата/время
- `date_picker` - Выбор даты
- `date_time_picker` - Дата и время
- `time_picker` - Время
- `color_picker` - Цвет

### Разметка (PRO)
- `message` - Сообщение
- `accordion` - Аккордеон
- `tab` - Вкладка
- `group` - Группа
- `repeater` - Повторитель
- `flexible_content` - Гибкий контент
- `clone` - Клон

## Location Rules (правила показа)

### По типу записи
```json
[{"param": "post_type", "operator": "==", "value": "product"}]
```

### По шаблону страницы
```json
[{"param": "page_template", "operator": "==", "value": "template-contact.php"}]
```

### По странице опций
```json
[{"param": "options_page", "operator": "==", "value": "site-settings"}]
```

### По таксономии
```json
[{"param": "taxonomy", "operator": "==", "value": "category"}]
```

### Комбинация условий (OR)
```json
[
  [{"param": "post_type", "operator": "==", "value": "post"}],
  [{"param": "post_type", "operator": "==", "value": "page"}]
]
```

### Комбинация условий (AND)
```json
[
  [
    {"param": "post_type", "operator": "==", "value": "product"},
    {"param": "post_taxonomy", "operator": "==", "value": "featured:yes"}
  ]
]
```
